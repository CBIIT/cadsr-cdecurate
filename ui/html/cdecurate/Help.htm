The Common Data Element (CDE) Curation Tool

The CDE Curation Tool allows content administrators to create, edit and manage the metadata within the caDSR meatdata repository. An account is required to use this tool, and you must have curatorial authority for your Context. This tool's features facilitate the use of the NCICB Enterprise Vocabulary System (EVS) to create administered component names and definitions, helping to ensure ISO/IEC 11179 compliance and also use of caDSR naming conventions.

CDE Curation Tool User's Guide will tell you how to:

  • Search the caDSR for Data Elements, Data Element Concepts, Value Domains, Conceptual Domains, Classification Scheme Items, Value/Meanings, Case Report Form (CRF) Questions, Permissible Values, Object Class, Property and Concepts
  • Curate (create and edit) Data Elements, Data Element Concepts and Value Domains both induvidually and in groups (block).
