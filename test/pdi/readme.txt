. Get the latest pom from http://maven-repository.com/artifact/pentaho-kettle/kettle-engine
. Rename the downloaded file into pom.xml
. Run "mvn install"
. Or if you just need the binary, download the .jar file :)

Notes:

Do not miss adding/include libswt/osx/swt.jar or libswt/osx64/swt.jar into your classpath if you are running on Mac.
